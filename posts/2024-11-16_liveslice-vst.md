# LiveSlice VST

I’m looking into beat slicers again. LiveSlice is an indie plugin that I thought had a lot of promise back when I first looked at it. I was interested in slicing audio as it came in and retriggering certain elements of it using midi. Not many slicers were designed to do this.

As a basic slicer LiveSlice is pretty cool but it focuses on triggering arrangements of the loaded loops. You can load a number of loops and then slice different sections of the loaded loops into arrangements, each of which can have multiple tracks. The track editor lets you do the normal slicer things like reverse a section or modify volume or send MIDI CCs per slice (this is actually pretty unique).

![](images/2024-11-16_liveslice-vst_image_2024-11-16_173654622.png)

At the bottom you have many arrangements that can be triggered with MIDI notes. The UI is a little bit fiddly but once you get used to it it’s pretty fun.

v 1.47 adds MIDI features that should be able to trigger recording.

![](images/2024-11-16_liveslice-vst_image_2024-11-16_174328280.png)

I’ll take another look at this one soon.
