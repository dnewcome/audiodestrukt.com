# Wavosaur &#8211; Replacement for Sound Forge

I have been looking for a free alternative to Sonic Foundry (now Sony) Sound Forge for a while now. I’m not too demanding since the last version that I really used was Sound Forge 6.0, before Sony bought the product.
I have tried to use the leading open source audio editor, Audacity, for a while now, but it has never clicked for me. The last straw is that one of the features that I use the most, markers, just seems to be broken. I can’t move markers around once placed, the keyboard shortcut doesn’t make sense, and I can’t seem to export them to separate files once I have them in place.
Finally I have found something that works kind of like Sound Forge used to work (at least the older versions, which is all I care about). This software is called Wavosaur. I can’t say I like the name too much, but looking through the menus, it is laid out a lot like Sound Forge was, and markers worked like a charm when I used it.
For a little background, one of the main things I use a standalone audio editor for is marking up drum breaks and exporting them as separate wav files to load into my sampler.
