# Microcontroller swarms

![](images/2012-05-20_microcontroller-swarms_banner_tinyavr.jpg)

What would you do with a microcontroller that is as small or smaller than an op-amp? I knew that Atmel had a series of tiny AVR chips but I didn’t realize that they went all the way down to a [6 pin device](http://www.atmel.com/devices/ATTINY10.aspx?tab=documents). This is smaller than most dual op-amp packages. 

I’ve had a few ideas for synth circuits that uses a few micros to function as wavetable oscillators. I think it would be perfectly feasible to get these chips working that way. Also, maybe you could have a few of these in a guitar pedal doing some bitcrushing or signal processing? These tiny chips do A/D and PWM so you could probably set one of these up to do some low-fi signal processing.
