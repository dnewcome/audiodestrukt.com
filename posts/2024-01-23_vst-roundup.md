# VST Roundup

Periodically I try out some new VSTs in my work. Sometimes I forget that I even tried something or I find a VST installed on my workstation and I can’t quite remember why I have it installed. This is an effort to document some of my experiments so I can re-trace the thread later.

I’ve been interested in arpeggiators for a long time and I found one recently called Stocha. [https://stochas.org/ss/](https://stochas.org/ss/)

This is a pretty standard arpeggiator VST plugin that has stochastic playback abilities. You can set probability weights to different steps and affect the playback in a statistically managed way.

![](images/2024-01-23_vst-roundup_ss_drum_poly.png)

So far so good. I will have to compare this to the PizPlz arpeggiator. Looks like the Piz website is down but you can still get the stuff from Google Drive apparently [https://code.google.com/archive/p/pizmidi/downloads](https://code.google.com/archive/p/pizmidi/downloads)
