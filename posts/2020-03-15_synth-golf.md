# Synth golf

I’m revisiting some ideas I had around minimal audio synthesizer circuits. One of the simplest methods of creating an oscillator is using a schmidt trigger oscillator using a 74LS14 inverter. The trick here is to get the oscillation range correct with the RC values. I used a 100uF cap and a 10K potentiometer here. I like to test these with some LEDs so I don’t have to listen to an audio drone for hours while I’m working on the circuit.

Another thing I keep forgetting is the range of C values when doing simple low pass filters. 50 or 100uF are good values for these. Smaller caps are useful for AC coupling etc.
